🌐 Lesson 0004：渠道管理——商品从哪里卖出去

前面三课讲了商品、库存、订单。现在的问题是：这些商品通过哪些渠道卖给谁？
渠道决定了定价、库存分配、结算方式和对账口径。渠道管不好，前面的都白搭。

1. 电商渠道的三类形态

渠道类型代表特点佣金/费用
品牌官网 insta360.com 自主可控，毛利最高，流量靠自己 无佣金，但需承担建站+推广成本
第三方平台 天猫、京东、Amazon、Shopee 流量大，规则多，受平台制约 佣金 5-15% + 广告费
分销/批发 经销商、B2B 平台 走量为主，定价权力低 批发折扣 + 返点
Insta360 的典型渠道组合

官网（品牌旗舰）+ Amazon（海外主力）+ 京东/天猫（国内主力）+ 线下经销商（全球）

这就是「全球电商业务」在渠道层面的真实画像。

2. 铺货：同一个 SKU 如何上到不同渠道

铺货 ≠ 创建新 SKU。铺货是在已有 SKU 的基础上，为每个渠道生成一条铺货记录：

📋 铺货记录的核心字段
{
  "sku_id":      "SKU20240001001",     // 引用已有 SKU
  "channel":     "JD",                  // 京东
  "channel_sku": "JD-X5-BLACK-128",     // 渠道侧的商品编码
  "price":       2999,                  // 渠道售价
  "stock_ratio": 0.3,                   // 分配 30% 库存给这个渠道
  "status":      "on_shelf"             // 上架/下架
}

🎬 例子

Insta360 X5 黑色 128GB 在深圳仓有 100 件库存。
渠道库存分配：

  • 官网：50 件（优先保障品牌体验）
  • 京东：30 件
  • Amazon US：20 件

这不是 3 个 SKU，是1 个 SKU × 3 条铺货记录。

3. 渠道定价策略

统一定价

所有渠道同价

✅ 品牌形象统一

❌ 平台促销难配合

适合：高端品牌

差异化定价

各渠道不同价

✅ 灵活配合平台活动

❌ 串货/比价风险

适合：多市场多币种

⚠️ 渠道运营的经典坑：窜货

窜货：经销商把 A 渠道的低价货拿到 B 渠道卖，破坏价格体系。

防范手段：渠道专属 SKU 编码、区域锁、经销商协议中的惩罚条款。

运营日常关注：各渠道实际成交价监控 → 异常低价预警。

4. 渠道结算——对账的核心

不同渠道的结算方式完全不同，这是运营做对账时必须搞清楚的：

渠道结算方式对账难点
官网直接收款（支付宝/微信/PayPal）支付手续费、退款原路返回
京东自营京东采购→京东卖给消费者采购价 vs 零售价、返利计算
天猫旗舰店消费者付给天猫→天猫结算给品牌佣金扣除、活动补贴
AmazonFBA / FBM 两种模式仓储费、配送费、广告费扣款
经销商批发合同、账期结算信用额度、逾期催收
🔑 对账核心公式
品牌实收 = 渠道销售额 
         − 平台佣金
         − 支付手续费
         − 推广费
         − 物流费（如 FBA）
         ± 退款/退货调整

运营对账就是逐笔核对这五个减项是否与平台账单一致。

5. 渠道绩效——怎么判断一个渠道好不好

指标公式健康值
渠道 GMV 占比渠道 GMV ÷ 总 GMV不过度依赖单一渠道（<50%）
渠道毛利率(收入−成本−渠道费用) ÷ 收入官网 > 平台 > 分销
退货率退货数 ÷ 销量不同品类不同，消费电子 2-5%
库存周转见 Lesson 0002分渠道看，平台通常快于官网
ROI渠道利润 ÷ 渠道投入> 1 即可，> 3 优秀

6. 测验

🧠 第 1 题

品牌官网毛利率 60%，天猫毛利率 40%，Amazon 毛利率 35%。为什么老板还是不肯砍掉低毛利的平台渠道？

A. 平台渠道虽然毛利低，但是流量和新客来源 B. 老板不懂数据，应该用数据说服他 C. 可以砍掉，把预算全投官网推广
✅ 正确。毛利不是唯一指标。平台渠道提供流量入口、品牌曝光、新客获取——这些是官网做不到的。很多用户是在 Amazon 搜到品牌 → 下次在官网买。
运营思维：看全链路价值，不只盯单一指标。
❌ 不是数据问题，是视角问题。只看毛利会忽略渠道的战略价值。
❌ 全部投官网推广 = 放弃站外流量。平台的搜索流量无法用广告完全替代。

7. 关键要点

#Takeaway
1三类渠道：官网（自有）、平台（天猫/京东/Amazon）、分销（经销商）
2铺货 = 1 个 SKU 在不同渠道生成铺货记录，不是新建 SKU
3不同渠道结算方式完全不同，对账逐一核对五个减项
4窜货是渠道管理的核心风险
5渠道绩效看综合指标，不只盯毛利率
