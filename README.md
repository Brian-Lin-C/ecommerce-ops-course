# 电商运营中台知识体系

7 节互动课程，从零入门电商运营核心业务。适合想入行电商运营 / 数据分析 / 业务中台的新人。

## 课程目录

| # | 课程 | 内容 |
|---|------|------|
| 01 | [商品、SPU、SKU、类目](lessons/0001-spu-sku-category.html) | 电商商品体系的三层基石 |
| 02 | [库存基础](lessons/0002-inventory-basics.html) | 入库出库、库存周转、安全库存 |
| 03 | [订单全链路](lessons/0003-order-lifecycle.html) | OMS、状态机、拆单、售后 |
| 04 | [渠道管理](lessons/0004-channels.html) | 官网/平台/分销、铺货、结算 |
| 05 | [销售核心指标](lessons/0005-sales-kpis.html) | GMV、转化率、客单价、拆解法 |
| 06 | [数据看板与对账](lessons/0006-dashboards-reconciliation.html) | SQL 模板、对账自动化、异常排查 |
| 07 | [AI Agent 重构电商运营](lessons/0007-ai-agent-ops.html) | Agent 七大应用场景与实操 |

## 特点

- **暗黑主题**，Tufte 风格排版，适合长时间阅读
- **每课含测验**，互动式选择题即时反馈
- **基于真实资料**，引用了阿里/京东/有赞/vivo 等电商中台实践
- **从零开始**，不需要任何电商基础

## 本地使用

直接用浏览器打开 `lessons/0001-spu-sku-category.html`，按顺序学习。
